# 📊 HR Attrition Analysis (Python + Power BI)

## 📌 Project Overview

Employee attrition is a critical challenge for organizations. In this project, I analyzed employee data using Python and built an interactive dashboard in Power BI to identify key factors contributing to attrition and provide actionable insights.

## 🎯 Objectives

* Understand employee attrition patterns
* Identify high-risk employee groups
* Provide data-driven recommendations to improve employee retention

## 🗂 Dataset

* HR Employee Attrition dataset
* Total Records: 1,470 employees
* Key features: Age, Department, Job Role, Monthly Income, Overtime, Job Satisfaction, Years at Company, etc.

## 🛠 Tools & Technologies

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Power BI
* DAX (Data Analysis Expressions)
* Power Query

## 🔍 Analysis Workflow

### 🔹 Data Analysis (Python)

* Data loading and exploration
* Data cleaning and preprocessing
* Feature engineering:

  * Attrition Flag
  * Salary Bands
  * Age Groups
  * Tenure Groups
* Exploratory Data Analysis (EDA)

### 🔹 Dashboard (Power BI)

* Built interactive dashboard
* Created DAX measures (Attrition Rate, Total Attrition)
* Visualized attrition across:

  * Department
  * Job Role
  * Salary Bands
  * Age Groups
  * Years at Company

## 📈 Key Insights

* Higher attrition observed among employees in lower salary bands
* Employees working overtime show significantly higher attrition
* Younger employees (26–35 age group) have higher turnover
* Majority attrition occurs within the first few years of employment

## 💡 Business Recommendations

* Improve salary structure for low-income employees
* Monitor and reduce excessive overtime
* Focus on retention during early career stages
* Improve work conditions in high attrition roles

## 🚀 Conclusion

This project demonstrates how combining Python-based analysis with Power BI visualization can help organizations understand attrition patterns and make data-driven HR decisions.
