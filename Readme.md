# HR Employee Attrition Analysis 📊

## 📌 Project Overview
Employee attrition is a critical issue for organizations. This project analyzes employee data to identify key factors contributing to attrition and provides actionable business insights.

## 🎯 Objectives
- Understand employee attrition patterns
- Identify high-risk employee groups
- Provide data-driven recommendations for retention

## 🗂 Dataset
- HR Employee Attrition dataset
- Records: 1,470 employees
- Features include Age, Department, Salary, Overtime, Job Satisfaction, Years at Company, etc.

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 🔍 Analysis Performed
- Data cleaning & preprocessing
- Feature engineering (AttritionFlag, Salary Bands, Age Groups)
- Exploratory Data Analysis (EDA)
- Attrition rate analysis across:
  - Departments
  - Salary Bands
  - Overtime
  - Age Groups
  - Tenure Groups

## 📈 Key Insights
- Higher attrition observed in lower salary bands
- Overtime employees show significantly higher attrition
- Younger employees experience higher turnover
- Attrition decreases with increased tenure

## 💡 Business Recommendations
- Revise salary structure for low-income bands
- Monitor and reduce excessive overtime
- Implement early-career retention programs
- Focus retention strategies on high-risk departments

## 🚀 Conclusion
This project demonstrates how data analysis can support HR decision-making and employee retention strategies.
